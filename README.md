#Libft

## Description
Libft is a foundational project at 42 school that involves re-coding standart c library funtions along with additional utility functions.

Projects goal is to create a personal, reusable library with useful functions while gaining understanding fundamental functions with effort.

## Instructions

### Compilation
The compile the entire library, run the following command in your terminal:
```bash
make
```
### Usage
After compilation include libft.h header in your source files and link the libft.a archive during compilation.
```bash
cc main -L. -lft
```
